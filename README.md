# 🕵️‍♂️ Zero-Login Hyper-Targeted Lead Extractor

An open-source, AI-powered intelligence tool designed to extract hyper-qualified leads from search engines without relying on platform logins, API keys, or paid proxies. 

Powered by **DeepSeek-R1** (via GitHub Models) and **DuckDuckGo**, this tool translates natural language into strict Boolean search queries, stealthily scrapes the web, and uses advanced reasoning to rigorously filter leads against your exact criteria.

## 🚀 Features
* **Zero Logins Required:** Never risk your personal LinkedIn or Twitter accounts. Operates entirely via public search indexing (X-Ray searching).
* **Massive AI Reasoning:** Uses the powerful DeepSeek-R1 model to evaluate and filter leads, ensuring zero false positives.
* **Stealth Operations:** Built-in human-mimicking delays to prevent IP bans.
* **100% Free to Run:** Utilizes the generous free tiers of GitHub Models and DuckDuckGo.
